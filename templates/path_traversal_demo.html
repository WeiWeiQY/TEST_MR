📁 路径遍历漏洞演示

⚠️ 危险环境 存在路径遍历漏洞，允许访问任意文件，请仅在授权环境测试。
易受攻击的代码： 
# 危险：直接拼接文件路径，未验证
filepath = os.path.join('data', filename)
with open(filepath) as f:
    content = f.read()
{% if content %}

文件内容：

{% if content is mapping %}{{ content | safe }}{% else %}{{ content }}{% endif %}
{% endif %}

POC测试示例：

1. 路径遍历读取敏感文件：

• ../../../../etc/passwd

• ../../data/files/readme.txt

2. URL编码绕过：

• %2e%2e%2fetc%2fpasswd

3. 使用绝对路径：

• /etc/passwd

修复方案：使用safe_join验证路径
