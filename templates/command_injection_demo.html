💡 测试提示

POC 测试示例：

1. 基本命令注入：127.0.0.1; whoami

2. 与操作符：127.0.0.1 && ls -la

3. 管道符：127.0.0.1 | cat /etc/passwd

4. 反引号：127.0.0.1 `cat /etc/passwd`

修复方案：

import re

import subprocess

# 验证输入

if not re.match(r'^[0-9.]+$', ip_address):

    raise ValueError('无效的IP地址')

# 不使用shell=True

subprocess.run(['ping', '-c', '3', ip_address])
